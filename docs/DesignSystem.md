# Design System Specifications

This guide details typography scales, color codes, and margins layout guidelines.

---

## 1. Typography Hierarchy

* **Display** (`text-display`): `Syne Bold`, 48px size, letter-spacing -0.03em. Used for covers and page welcome metrics.
* **H1** (`text-h1`): `Syne SemiBold`, 40px size, letter-spacing -0.02em.
* **H2** (`text-h2`): `Syne SemiBold`, 32px size. Used for section headers.
* **H3** (`text-h3`): `Syne SemiBold`, 24px size.
* **Card Title** (`text-card-title`): `Syne SemiBold`, 20px size.
* **Body** (`text-body`): `DM Sans Regular`, 16px size. Used for descriptions.
* **Small** (`text-small`): `DM Sans Regular`, 14px size.
* **Caption** (`text-caption`): `DM Sans Regular`, 12px size.
* **Label** (`text-label`): `DM Sans Medium`, 11px size, uppercase, letter-spacing 0.05em.

---

## 2. Spacing Grid (8pt System)
All margins and padding heights are constrained to multiples of 8:
* `xs`: 8px
* `sm`: 12px
* `md`: 16px
* `lg`: 24px
* `xl`: 32px
* `2xl`: 48px

---

## 3. Standard Color Tokens
* **Background** (`--bg-primary`): `#0D1512`
* **Surface** (`--bg-secondary`): `#141E18`
* **Surface Hover** (`--bg-hover`): `#1B2821`
* **Border** (`--border`): `#223128`
* **Primary Accent** (`--accent`): `#7BAE7F`
